Разработка грамматики

Определим грамматику комментариев языка Python G[<C>] в нотации Хомского с продукциями P:

1.  <C>       → <SHARP>
2.  <C>       → <FOSQ>
3.  <C>       → <FODQ>
4.  <SHARP>   → '#' <SLC> <newline>
5.  <SLC>     → <symbol> <SLC> | ε
6.  <FOSQ>    → ''' <BODY_SQ>
7.  <BODY_SQ> → <symbol> <BODY_SQ> | '''
8.  <FODQ>    → """ <BODY_DQ>
9.  <BODY_DQ> → <symbol> <BODY_DQ> | """

Где:

  • <C> — стартовый нетерминал (комментарий любой формы).
  • <SHARP> — однострочный комментарий # ….
  • <SLC> — тело однострочного комментария (любые символы до конца строки).
  • <FOSQ> — открывающий тройной одинарный апостроф '''.
  • <BODY_SQ> — тело многострочного комментария в одинарных кавычках.
  • <FODQ> — открывающий тройной двойной кавычки """.
  • <BODY_DQ> — тело многострочного комментария в двойных кавычках.
  • <symbol> — любой символ, кроме одиночной кавычки, двойной кавычки и перевода строки.
  • ε — пустая строка (конец однострочного комментария).

Следуя введённому формальному определению грамматики, представим G[<C>] её составляющими:

  • Z (начальный символ): <C>
  • VT (терминалы): { '#', ''' , """, <symbol>, <newline> }
  • VN (нетерминалы): { <C>, <SHARP>, <SLC>, <FOSQ>, <BODY_SQ>, <FODQ>, <BODY_DQ> }
